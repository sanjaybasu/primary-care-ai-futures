# Results Directory

This directory contains outputs generated by running the analysis pipeline. It is initially empty and will be populated when you execute the notebooks.

## Directory Structure

After running the full pipeline, this directory will contain:

```
results/
├── figures/            # Publication-quality figures (PNG, 300+ DPI)
│   ├── figure1_trends_svi.png
│   ├── figure2_simulation_scenarios.png
│   ├── figure3_cost_effectiveness.png
│   ├── figure_s1_calibration.png
│   ├── figure_s2_hosp_sensitivity.png
│   ├── figure_s3_prisma.png
│   ├── figure_s4_digital_map.png
│   └── figure_s5_monte_carlo.png
├── tables/             # CSV tables for manuscript
│   ├── table1_temporal_validation.csv
│   ├── table2_cost_effectiveness.csv
│   └── supplementary_tables.csv
├── validation/         # Temporal validation outputs
│   ├── calibration_metrics.json
│   ├── prediction_df.csv
│   └── 03_temporal_validation.ipynb (executed)
└── models/             # Trained model weights
    └── rssm_best_model.pt
```

## How to Generate Results

Run the full pipeline:

```bash
# Option 1: Run individual notebooks in sequence
jupyter notebook notebooks/01_comprehensive_trends.ipynb
jupyter notebook notebooks/02_rssm_science_sim.ipynb
jupyter notebook notebooks/03_temporal_validation.ipynb
jupyter notebook notebooks/04_cost_effectiveness.ipynb

# Option 2: Automated execution
python src/reproduce_results.py

# Option 3: Via CodeOcean capsule
bash codeocean/run_capsule.sh
```

## Expected Outputs

See `docs/expected_outputs.json` for the numerical values you should obtain. Your results should match within ±10% due to Monte Carlo simulation random seed variation.

## File Size

Total size after generation: ~50 MB
- Figures: ~4 MB (8 PNG files)
- Tables: ~1 MB (CSV files)
- Models: ~45 MB (PyTorch checkpoint)
